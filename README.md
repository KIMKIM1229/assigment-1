## 安裝與使用

1. **安裝依賴**：
   - 確保你的系統已經安裝了 Node.js 和 Ionic CLI。
   - 在項目根目錄運行以下命令來安裝依賴：
     ```bash
     npm install
     ```

2. **運行應用**：
   - 在項目根目錄運行以下命令來啟動應用：
     ```bash
     ionic serve
     ```
   - 打開瀏覽器，訪問 `http://localhost:8100` 查看應用。

## 開發環境設置

- **Ionic Framework**：用於構建移動應用和桌面網頁應用。
- **JavaScript (ES6+)**：用於編寫應用邏輯和處理事件。
- **CSS / SCSS**：用於設計和樣式。
- **HTML**：用於結構化應用內容。
